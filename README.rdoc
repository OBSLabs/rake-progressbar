= rake-progressbar

Progress bar for long runnung jobs in terminal - useful mostly for rake tasks (imports, exports, archives etc.)

== Usage

	bar = RakeProgressbar.new(0)
	100.times do 
		bar.inc
	end
	bar.finished
	
It will generate:
 25.2% [#####_______________] 01:02:30 -> 00:25:13
